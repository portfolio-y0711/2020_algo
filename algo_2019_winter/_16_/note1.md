##### 예제 1

```ts

    5
   / \
  3   6
 / \   \
2   4   7

tree:
[2]
[2, 3]
[2, 3, 4]
[2, 3, 4, 5]
[2, 3, 4, 5, 6]
[2, 3, 4, 5, 6, 7]

6C2 = (6 x 5) / (2 x 1) = 15
comb:
[2, 3] [2, 4] [2, 5] [2, 6] [2, 7]
-> 5, 6, 7, 8, 9
[3, 4] [3, 5] [3, 6] [3, 7]
-> 7, 8, 9, 10
[4, 5] [4, 6] [4, 7]
-> 9, 10, 11
[5, 6] [5, 7] 
-> 11, 12
[6, 7]
-> 13

```
